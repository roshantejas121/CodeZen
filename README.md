# 🚀 CodeZen: The Elite Engineering Command Center

**CodeZen** is a high-performance, professional-grade platform designed to accelerate technical mastery. It merges the structured curriculum of W3Schools with the high-intensity execution environment of LeetCode, all powered by an integrated AI Technical Audit system.

## 🎯 The Challenge

> **Problem Statement**: Constant context switching between coding, debugging, and scattered tools breaks developer flow, reducing productivity and effective learning.

**CodeZen** solves this by providing a **Unified Engineering Ecosystem**. Instead of jumping between tabs for documentation, compilers, and GitHub, developers can stay in a "Deep Flow" state where learning and execution happen in a single, high-fidelity workspace.

## 🌟 Core Pillars

### 1. The W3Schools-Style Academy
A high-density technical library featuring over **50+ chapters** across HTML, CSS, JavaScript, Python, and SQL. 
*   **Triple-Pane Layout**: Curriculum (Left) → Deep Documentation (Center) → Live IDE (Right).
*   **Sequential Learning**: Guided "Next/Previous" roadmap for foundational mastery.

### 2. Universal Polyglot Compiler
A "real-to-the-core" execution engine providing genuine server-side compilation for multiple technical domains.
*   **Genuine Execution**: Powered by the **Piston API** (GCC, Node.js, Python 3.10, OpenJDK).
*   **Real Console**: Live `stdout` and `stderr` feedback in a professional Linux-styled terminal.
*   **Smart Preview**: Real-time visual rendering for HTML/CSS/JS with "Smart Injector" technology.

### 3. GitHub Intelligence Hub
A professional discovery center for open-source research and technical evaluation.
*   **AI Technical Audit**: Simulate deep-dives into any repository for security, quality, and performance.
*   **Discovery Engine**: Explore high-impact repositories with real-time star/fork metrics and trending tech hubs.

### 4. Organized Command Center
A streamlined, high-density dashboard for tracking project objectives and engineering growth.
*   **Level Progress**: XP-based mastery tracking across 50+ technical tracks.
*   **Productivity Suite**: Integrated technical news and intelligence feeds.

## 🛠️ Technical Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/) (Strict Mode)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: Vanilla CSS & Glassmorphic UI Tokens
- **Icons**: [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)
- **API**: Custom Serverless Routes & Piston Execution Engine

## 🚀 Quick Start

### 1. Clone the Repository
```bash
git clone https://github.com/roshantejas121/CodeZen.git
cd CodeZen
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run Development Server
```bash
npm run dev
```

### 4. Access the Hub
Open [http://localhost:3000](http://localhost:3000) to enter the Command Center.
